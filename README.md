# [Swift Package Manager](#SPM)

1. **Add a Swift Package File**
   - In your Xcode project, go to **File** > **Swift Packages** > **Add Package Dependency**.

2. **Enter Package Repository URL**
   - In the dialog that appears, enter the URL of the repository that hosts the CompiletSDK package:

```ruby
    https://github.com/compilet-dev/ios-compilet-sdk
```

**Note:** Please select the appropriate version of the SDK for your project, then click Next to proceed and confirm the integration.
